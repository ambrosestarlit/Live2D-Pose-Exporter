取扱説明

このタブでできること

作成済みLive2Dモデルと画像素材を読み込み、タイムライン上でキーフレームを入力して、連番PNG ZIPとして書き出します。モデル制作やメッシュ編集ではなく、完成済みモデルを使ったアニメーション作成用です。

1. 初回準備

  1. Live2D Cubism SDK for Web から live2dcubismcore.min.js を用意します。
  2. アプリの vendor フォルダ内に live2dcubismcore.min.js を配置します。
  3. index.html をブラウザで開きます。Coreが検出されると、モデル読み込みが可能になります。

Cubism Coreが見つからない場合は、ファイル名と配置場所が vendor/live2dcubismcore.min.js になっているか確認してください。

2. モデルを読み込む

  1. 左パネルの モデルフォルダを読み込み、または モデルZIPを読み込み を押します。
  2. model3.json を含むLive2Dモデルを選択します。
  3. 検出モデル に表示されたモデルを選び、選択モデルを追加 を押します。
  4. 複数キャラを使う場合は同じ手順で追加します。キャラ一覧で上にあるキャラほど前面に表示されます。

3. 音声・画像素材を読み込む

  • 音声を読み込み：音声を再生しながら、タイミングに合わせてキーを打てます。
  • 画像を読み込み：背景、小物、エフェクト素材などを追加できます。
  • 画像素材は、移動・回転・拡大縮小・不透明度・表示ON/OFFをキーフレーム化できます。
  • 画像レイヤーもタイムラインに表示され、上にあるレイヤーほど前面に描画されます。

4. タイムラインの基本

  • 長さ(秒)：アニメーション全体の長さです。連番PNG書き出しもこの長さを基準にします。
  • FPS：1秒あたりのフレーム数です。例：5秒 / 24FPS の場合、約120枚を書き出します。
  • 現在秒：現在のシークバー位置です。直接数値入力もできます。
  • 自動キー：ONの状態でスライダーやドラッグ操作をすると、現在秒にキーフレームが入力されます。
  • 前キー / 次キー：選択中レイヤーまたは選択中トラックの前後のキーへ移動します。

5. キーフレーム入力

  1. シークバーを、動きを入れたい時刻へ合わせます。
  2. モデルのパラメータ、モデル位置、画像位置などを調整します。
  3. 自動キーONなら、その時刻にキーフレームが保存されます。
  4. 自動キーOFFの場合は、調整したいトラックを選択して キーフレーム追加 を押します。

キーフレームはパラメーターごとのトラックで管理されます。たとえば、目だけ途中で変更したい場合は、目のパラメータートラックだけにキーを追加できます。

6. キーフレーム編集

  • タイムライン上の丸いキーをクリックすると、右パネルの キーフレーム設定プロパティ タブが開きます。
  • キー位置(秒)：キーの時刻を後から数値で変更できます。
  • 値：選択キーの内容を数値で直接調整できます。
  • 補間：なめらか は前後のキーを補間し、次キーまで固定 は次のキーまで同じ値を維持します。
  • 現在の状態で上書き：現在のプレビュー状態を、選択中のキーへ上書きします。
  • キー削除：選択中のキーを削除します。プレビュー下のボタンからも実行できます。
  • キー複製：選択中のキーを複製し、次の1フレーム位置へ追加します。プレビュー下のボタンからも実行できます。
  • タイムライン上のキーはドラッグで左右に移動できます。

7. ループ機能

  • キーフレーム設定プロパティの ループ をONにすると、そのトラックの入力済みキーがタイムライン終了まで繰り返し再生されます。
  • 書き出し時も、ループONのトラックは書き出し時間いっぱいループします。
  • ループ挿入 ボタンでは、指定したトラックのキー列を実際のキーフレームとして複製挿入できます。
  • ループ挿入は、ループ回数を入力し、対象トラックを選んで ループする を押します。
  • ループ挿入はキーを増やす処理なので、後から個別に調整したい揺れや反復動作に向いています。

8. 髪揺れ自動入力

  1. シークバーを、髪揺れを開始したい位置へ合わせます。
  2. 髪揺れ自動入力 を押します。
  3. プリセットから 普通の風 / そよ風 / 強風 / 軽く弾む / 大きく弾む を選びます。
  4. 揺れの回数、強さ、秒数をスライダーまたは数値入力で調整します。
  5. 対象パラメーターを選び、挿入 を押すと、複数のキーフレームが自動入力されます。

髪候補は Hair、髪、sway、physics などの名前を含むパラメーターから自動検出します。モデルによって名称が違うため、必要に応じて手動で対象を選んでください。

9. 連番PNG ZIP書き出し

  1. 左パネルでキャンバスサイズ、背景あり/なし、背景色を設定します。
  2. タイムラインの長さとFPSを確認します。
  3. 連番PNG ZIP書き出し を押します。
  4. 0001.png、0002.png のような連番PNGがZIPで保存されます。

市松背景は確認用です。透過PNGで書き出したい場合は、背景ありで書き出すをOFFにしてください。

このタブでできること

タイムラインを使わず、Live2Dモデルのポーズや表情を調整して、現在の状態をPNGとして保存できます。SNS投稿用の立ち絵、表情差分、サムネ素材、静止画素材の作成に向いています。

1. モデルを配置する

  1. モデルフォルダまたはZIPを読み込み、選択モデルを追加 します。
  2. キャラ一覧から操作したいキャラを選びます。
  3. プレビュー上部の モデル移動、モデルサイズ を使って配置を調整します。
  4. 左パネルの X / Y / 拡大率 でも数値調整できます。

2. 表情・ポーズを作る

  • 顔の向き：顔の左右・上下の向きを調整します。
  • 顔の傾き：首をかしげるような傾きを調整します。
  • 視線移動：目線の向きを調整します。
  • まばたき：目の開閉を調整します。
  • 笑顔：笑顔系パラメーターをまとめて調整します。
  • 口の開閉：口の開き具合を調整します。
  • 眉毛の表情：困り眉、怒り眉などの表情調整に使います。
  • 右パネルの パラメータ から、各パラメーターを個別に調整することもできます。

3. Live2D機能のON/OFF

  • 待機モーションON：モデルに設定されている待機モーションを使います。
  • 自動呼吸ON：呼吸のような自然な動きを加えます。
  • 物理演算ON：髪や服など、モデル側で設定済みの物理演算を反映します。
  • 自動まばたきON：自動でまばたきを行います。

静止画として完全に固定したい場合は、待機モーション・自動呼吸・物理演算・自動まばたきをOFFにしてから保存すると安定します。

4. 画像素材を重ねる

  • 画像を読み込み からPNG/JPG/WebPなどを追加できます。
  • 右パネルの 画像調整 で、X/Y、拡大率、回転、不透明度を調整します。
  • キャラや画像は、一覧の上にあるものほど前面に表示されます。
  • 画像素材を背景や装飾として使う場合は、レイヤー順を調整してください。

5. シーンプリセット

  • 現在のキャラ位置、パラメーター、表情、画像配置などを シーンプリセット として保存できます。
  • プリセット名を入力して 保存 を押すと一覧に追加されます。
  • 保存したプリセットを適用すると、その状態へ戻せます。
  • 複数プリセットを選択して、差分画像をまとめてPNG ZIP保存できます。
  • JSON保存/読込を使うと、プリセット設定を別ファイルとして保管できます。

6. 現在フレームをPNG保存

  1. 左パネルでキャンバスサイズを設定します。
  2. 必要なら 背景ありで書き出す をONにし、背景色を選びます。
  3. 透過PNGにしたい場合は、背景ありで書き出すをOFFにします。
  4. 現在フレームをPNG保存 を押すと、プレビュー中の状態を1枚のPNGとして保存します。

7. キャンバス設定

  • 幅 / 高さ：書き出し画像のサイズを指定します。
  • 原寸：選択中モデルの原寸サイズを取得できる場合、そのサイズをキャンバスに反映します。
  • 1000×1000 / 1920×1080 / 1080×1920：よく使うサイズのプリセットです。
  • キャンバスサイズ反映 を押すと、入力したサイズがプレビューと書き出しに反映されます。

8. 注意点

  • このアプリはLive2Dモデルそのものを作るアプリではありません。作成済みの model3.json 対応モデルを使います。
  • モデルごとにパラメーター名が違うため、一部の便利ツールが反応しない場合があります。その場合は右パネルのパラメータ一覧から手動調整してください。
  • ブラウザ上で動作するため、大きなモデルや長い連番PNG書き出しでは処理に時間がかかることがあります。
  • 市松背景はプレビュー確認用で、背景あり書き出しをONにしない限りPNGには入りません。
