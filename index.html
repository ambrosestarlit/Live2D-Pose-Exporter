取扱説明

  1. Live2D Cubism SDK for Web から live2dcubismcore.min.js を取得し、vendor フォルダへ配置します。
  2. model3.json を含むモデルフォルダ、またはZIPを読み込みます。
  3. パラメータスライダー、表情、モーション、待機モーション、自動呼吸、物理演算、自動まばたきを調整します。
  4. 現在状態をプリセット保存できます。複数プリセットを選んでPNG ZIP書き出しできます。
  5. 市松背景は確認用です。PNG出力は透明背景です。

Live2Dモデルの制作・メッシュ編集は行わず、作成済みモデルの表示とパラメータ操作に特化しています。
