# Live2D Animation Studio

作成済みLive2Dモデルを読み込み、ブラウザ上でキーフレームアニメーションや静止画ポーズを作成し、PNGとして書き出すツールです。

このアプリは、Live2Dモデルを作るアプリではありません。Cubismなどで作成済みの `model3.json` 対応モデルを読み込み、アニメーション・ポーズ・画像素材の配置を行うためのアプリです。

---

## 現行版の主な機能

- 作成済みLive2Dモデルの読み込み
- モデルフォルダ読み込み
- モデルZIP読み込み
- 複数キャラ配置
- キャラの表示順変更
- Live2Dパラメーターの調整
- 表情・モーション再生
- 待機モーション / 自動呼吸 / 物理演算 / 自動まばたき ON/OFF
- 音声読み込み
- 画像素材読み込み
- 画像素材の移動 / 回転 / 拡大縮小 / 不透明度調整
- タイムライン編集
- パラメーターごとのキーフレーム管理
- キーフレームの選択 / 移動 / 数値編集 / 削除 / 複製
- トラック単位のループ再生
- ループ挿入によるキーフレーム複製
- 髪揺れ自動入力
- 現在フレームPNG保存
- 連番PNG ZIP書き出し
- シーンプリセット保存 / 読込 / PNG ZIP書き出し
- 日本語 / 英語 / 韓国語UI切り替え

---

## 初回準備

1. Live2D Cubism SDK for Web から `live2dcubismcore.min.js` を用意します。
2. アプリ内の `vendor` フォルダへ配置します。
3. 配置パスは以下の形にしてください。

```text
vendor/live2dcubismcore.min.js
```

4. `index.html` をブラウザで開きます。
5. 画面左上のCoreステータスで、Cubism Coreが検出されていることを確認します。

Coreが見つからない場合は、ファイル名、配置場所、大文字小文字を確認してください。

# アニメーション制作

## 1. モデルを読み込む

1. 左パネルの「モデルフォルダを読み込み」または「モデルZIPを読み込み」を押します。
2. `model3.json` を含むLive2Dモデルを選択します。
3. 「検出モデル」に表示されたモデルを選びます。
4. 「選択モデルを追加」を押します。
5. 複数キャラを使う場合は、同じ手順で追加します。

キャラ一覧では、上にあるキャラほど前面に表示されます。

## 2. 音声を読み込む

1. 左パネルの「音声を読み込み」を押します。
2. 音声ファイルを選択します。
3. 音声プレイヤーで再生しながら、シークバー位置を確認できます。
4. 音声に合わせてパラメーターや画像位置を調整すると、タイミング合わせがしやすくなります。

## 3. 画像素材を読み込む

1. 左パネルの「画像を読み込み」を押します。
2. PNG / JPG / WebP などの画像を選択します。
3. 右パネルの「画像調整」で、X / Y / 拡大率 / 回転 / 不透明度を調整します。
4. 画像素材もキーフレームアニメーションに対応しています。

画像素材は、背景、小物、演出パーツ、エフェクト素材などに利用できます。

## 4. タイムライン設定

プレビュー下にある操作列で、タイムライン全体を設定します。

- **再生**：タイムラインを再生します。
- **停止**：再生を停止します。
- **キーフレーム追加**：選択中または直前に編集したトラックへキーを追加します。
- **キー削除**：選択中のキーフレームを削除します。
- **キー複製**：選択中のキーフレームを次の1フレーム位置へ複製します。
- **前キー / 次キー**：前後のキーフレームへ移動します。
- **髪揺れ自動入力**：現在位置から髪揺れキーを自動挿入します。
- **ループ挿入**：選択したトラックのキー列を指定回数分複製します。
- **長さ(秒)**：アニメーション全体の長さです。
- **FPS**：1秒あたりのフレーム数です。
- **現在秒**：現在の再生位置です。数値入力もできます。
- **自動キー**：ONの状態で操作すると、現在秒にキーが自動入力されます。

## 5. キーフレームを入力する

1. シークバーを動かしたい時刻へ合わせます。
2. モデルのパラメーター、モデル位置、画像位置などを調整します。
3. 自動キーONの場合、調整した内容が現在秒にキーフレームとして保存されます。
4. 自動キーOFFの場合は、調整したいトラックを選択し、「キーフレーム追加」を押します。

キーフレームは、Live2Dモデル全体ではなく、パラメーターごとのトラックで管理されます。

例：

- 目だけ途中で閉じる
- 口だけ途中で開ける
- 顔の角度だけ別タイミングで変える
- 画像の不透明度だけフェードさせる

このような個別編集ができます。

## 6. キーフレームを編集する

1. タイムライン上の丸いキーをクリックします。
2. 右パネルが「キーフレーム設定プロパティ」タブへ切り替わります。
3. 以下の項目を編集できます。

- **キー位置(秒)**：キーフレームの時間を数値で変更します。
- **補間**：キー間の動き方を選びます。
  - **なめらか**：前後のキーを補間して動きます。
  - **次キーまで固定**：次のキーまで同じ値を保ちます。
- **値**：選択中トラックの値を直接編集します。
- **現在の状態で上書き**：現在のプレビュー状態で選択キーを上書きします。
- **キー削除**：選択キーを削除します。
- **キー複製**：選択キーを複製します。
- **ループ**：そのトラックのキー列をタイムライン終了まで繰り返します。

タイムライン上のキーは、ドラッグで左右へ移動できます。

## 7. ループ機能

ループには2種類あります。

### トラックのループ再生

キーフレーム設定プロパティの「ループ」をONにすると、そのトラックの入力済みキーがタイムライン終了まで繰り返し再生されます。

このループは、連番PNG書き出し時にも反映されます。

向いている用途：

- 髪のゆらゆら
- 目の反復動作
- 小物の揺れ
- 光やエフェクトの周期動作

### ループ挿入

「ループ挿入」ボタンを押すと、ポップアップが開きます。

1. ループ回数を入力します。
2. ループさせたいキーがあるパラメーターを選択します。
3. 「ループする」を押します。
4. 選択したトラックのキー列が、指定回数分複製されて挿入されます。

ループ挿入は、実際のキーフレームを増やす処理です。挿入後に、個別のキーを調整したい場合に向いています。

## 8. 髪揺れ自動入力

1. シークバーを、髪揺れを開始したい位置へ合わせます。
2. 「髪揺れ自動入力」を押します。
3. プリセットを選びます。

プリセット：

- 普通の風
- そよ風
- 強風
- 軽く弾む
- 大きく弾む

4. 揺れの回数、強さ、揺れアニメーションの秒数を調整します。
5. 対象パラメーターを選択します。
6. 「挿入」を押します。

髪候補は、`Hair`、`髪`、`揺れ`、`sway`、`yure`、`physics` などを含むパラメーターから自動検出されます。

モデルによってパラメーター名が違うため、候補に出ない場合は手動で対象を選んでください。

## 9. 連番PNG ZIP書き出し

1. 左パネルでキャンバスサイズを設定します。
2. 背景あり/なしを確認します。
3. タイムラインの長さとFPSを確認します。
4. 「連番PNG ZIP書き出し」を押します。
5. `0001.png`、`0002.png` のような連番PNGがZIPで保存されます。

透過PNGで書き出したい場合は、「背景ありで書き出す」をOFFにしてください。

# 静止画・ポーズ制作

## 1. モデルを配置する

1. モデルを読み込みます。
2. キャラ一覧から操作したいキャラを選びます。
3. プレビュー上部の「モデル移動」「モデルサイズ」を使って配置します。
4. 左パネルの X / Y / 拡大率 でも数値調整できます。

## 2. 表情やポーズを作る

プレビュー上部の操作ツールを使うと、よく使うパラメーターをまとめて調整できます。

- **顔の向き**：顔の上下左右を調整します。
- **顔の傾き**：首をかしげるような傾きを調整します。
- **視線移動**：目線の向きを調整します。
- **まばたき**：目の開閉を調整します。
- **笑顔**：笑顔系パラメーターを調整します。
- **口の開閉**：口の開き具合を調整します。
- **眉毛の表情**：眉の形や表情を調整します。

右パネルの「パラメータ」一覧から、個別のパラメーターを直接調整することもできます。

## 3. Live2D機能のON/OFF

右パネルの「Live2D操作」で、以下を切り替えできます。

- **待機モーションON**
- **自動呼吸ON**
- **物理演算ON**
- **自動まばたきON**

静止画として完全に固定したい場合は、これらをOFFにすると安定します。

## 4. 画像素材を重ねる

1. 「画像を読み込み」から画像素材を追加します。
2. 右パネルの「画像調整」で、位置・拡大率・回転・不透明度を調整します。
3. レイヤー順を変更して、キャラや素材の前後関係を調整します。

画像素材は、背景、装飾、エフェクト、小物などに使用できます。

## 5. シーンプリセットを使う

現在の状態をシーンプリセットとして保存できます。

保存される内容の例：

- キャラ位置
- キャラ拡大率
- 表情
- パラメーター値
- 画像素材の位置
- 画像素材の回転
- 画像素材の不透明度

使い方：

1. プリセット名を入力します。
2. 「保存」を押します。
3. 一覧に保存されます。
4. 保存したプリセットを適用すると、その状態へ戻せます。
5. 複数プリセットを選択して、PNG ZIPとしてまとめて保存できます。

JSON保存/読込を使うと、プリセット設定を別ファイルとして保管できます。

## 6. 現在フレームをPNG保存

1. キャンバスサイズを設定します。
2. 透過PNGにしたい場合は「背景ありで書き出す」をOFFにします。
3. 背景色を入れたい場合は「背景ありで書き出す」をONにして、背景色を選びます。
4. 「現在フレームをPNG保存」を押します。

現在プレビューされている状態が1枚のPNGとして保存されます。

## 7. キャンバス設定

左パネルの「キャンバス / 書き出し」で設定します。

- **幅 / 高さ**：書き出しサイズを指定します。
- **原寸**：選択中モデルの原寸サイズを取得できる場合に反映します。
- **1000×1000**：正方形素材向けです。
- **1920×1080**：横長動画・サムネ向けです。
- **1080×1920**：縦長動画・スマホ向けです。
- **キャンバスサイズ選択プルダウン / キャンバスサイズ反映**：入力したサイズを適用します。
- **市松背景を表示**：透過確認用の表示です。書き出しには含まれません。
- **背景ありで書き出す**：ONにすると指定色の背景を含めます。

## 注意点

- このアプリはLive2Dモデル制作アプリではありません。
- `model3.json` を含む作成済みLive2Dモデルが必要です。
- モデルごとにパラメーター名が違うため、一部の操作ツールや髪揺れ候補が反応しない場合があります。
- その場合は、右パネルのパラメータ一覧から手動で調整してください。
- 大きなモデルや長い連番PNG書き出しでは、ブラウザの処理に時間がかかることがあります。
- 透過PNGを書き出したい場合は、「背景ありで書き出す」をOFFにしてください。

## 今回の更新：シーン管理・プロジェクト保存・絞り込みポップアップ

### シーン管理
左パネル最上部に「シーン管理 / プロジェクト」を追加しました。

- `Scene選択`：作業するシーンを切り替えます。切り替え前に現在の状態が自動保存されます。
- `シーン追加`：現在の状態をもとに新しいシーンを追加します。
- `シーン削除`：現在選択しているシーンを削除します。最後の1シーンは削除できません。

各シーンには、モデルの位置・表示状態・パラメーター・画像素材・レイヤー順・タイムライン設定・キーフレーム・ループ設定・絞り込み状態が保存されます。

### プロジェクトJSON保存
`プロジェクトJSON保存` を押すと、以下をまとめたJSONを保存できます。

- シーン一覧
- 現在選択中のシーン
- Live2Dモデルの仮想ファイルデータ
- 画像素材データ
- 音声データ
- タイムライン設定
- キーフレーム情報
- シーンプリセット

`プロジェクトJSON読込` から保存済みJSONを読み込むと、モデル・画像・音声・シーン情報を復元します。

### タイムライン絞り込み
タイムラインのモデル一覧最上部にある `表示トラック` ボタンを押すと、ポップアップで表示するキーフレーム項目を選べます。

- チェックが0個：入力済みキーフレームのある項目をすべて表示
- チェックが1個以上：チェックした項目のみ表示

ドロップダウンではなくポップアップ化したため、項目数が多い場合でも下部が見切れにくくなっています。
